Declared as public node
3.1.1.1. Comments and Information
All_Doors_false Comments:
This module is part of the monitoring of the door opening/closing sequence. More specifically to indicate an error if all the sensors are not false as expected after a certain amount of time. It can be instanciated to monitor both the opening and closing sequence.
